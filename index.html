Voice and Tone

Every word Kneuralabs publishes must reinforce trust, precision, and authority. We do not hedge. We do not hype.

TraitInstruction
AuthoritativeState positions directly. We are the expert in the room.
PreciseUse specific numbers, frameworks, and examples. No vague generalisations.
ClearPlain English. Remove jargon unless addressing a technical audience.
ConfidentRemove hedging words entirely: might, perhaps, kind of, sort of, basically.
ActiveActive voice only. We delivered X — not X was delivered by us.
StructuralOrganised and logical. Every paragraph earns its place.

Colour Rules

D OUse Electric Cyan (#00F0FF) as the sole accent colour for CTAs, rule lines, and active states.
D OUse Charcoal (#141414) for H1 headings and all primary display text.
D OUse Deep Navy (#204376) for H2, borders, and structural document elements.
D OKeep page backgrounds White. Smoke (#EEF1F5) for alternating rows only.
D OUse Logo Art colours (Teal, Crimson, Electric Blue) only in charts and diagrams.
D OEnsure WCAG AA contrast: 4.5:1 for body text, 3:1 for large display text.
D O N TUse more than 3 colours on any single page or slide.
D O N TUse any blue, teal, or cyan not defined in this palette.
D O N TUse brand Crimson (#C5062A) to indicate errors — use Alert Red (#E53E3E) instead.
D O N TPlace Electric Cyan text on white backgrounds for body copy — insufficient contrast.
D O N TUse Semantic colours (Emerald, Alert Red, Amber) for decoration or branding.
W A R NCyan (#00F0FF) is accent only. It fails WCAG AA at small sizes on white. Always pair with dark text for legibility.

AI Tool Instructions — Logo Handling

A ILogo file must always have a transparent background (PNG with alpha channel). Never place a white, black, or coloured rectangle behind the art mark.
A IWord document header: set to 22 × 22 px. Set text wrapping to In Line With Text. No floating placement.
A IPowerPoint title slide: 120 × 120 px. Art mark top-left. Text logo (KNEURALABS in Inter Bold White) sits to the right, vertically centred.
A IHTML: use an <img> tag with alt='Kneuralabs logo'. Set height via var(--kn-logo-height-*). Width must be auto. Never fix both dimensions.
D OPreserve the 1:1 aspect ratio of the art mark at all times.
D OUse PNG format for all digital logo files. Use EPS or SVG for print applications.
D O N TRecreate the art mark geometry by hand — use the approved logo file only.
D O N TCrop, mask, rotate, flip, or apply any visual effect to the art mark.
D O N TAdd drop shadows, glows, outlines, or any effects to any logo component.
W A R NA JPEG logo always carries a visible background rectangle. JPEG has no transparency support. Reject any JPEG logo file and request the PNG version.

Typography Rules

D OUse Inter as the primary font. Arial is the fallback when Inter is unavailable.
D OUse JetBrains Mono exclusively for hex codes, file names, version numbers, and code.
D OSet body font to 10pt in Word and Excel. 18pt minimum in PowerPoint.
D OSentence case for all body paragraphs. Title case for all headings H1 through H4.
D OKNEURALABS is always written in ALL CAPS in every heading, logo context, document header, slide title, and text logo application — no exceptions.
D OLine spacing 1.15 for body. 6pt space after each paragraph.
D OBold reserved for headings and key terms only — never whole paragraphs.
D OUnderline reserved for hyperlinks only.
D O N TUse more than 2 font families in any single document.
D O N TApply italics — restructure the sentence to carry emphasis instead.
D O N TUse ALL CAPS in body copy — reserved for the wordmark only.

Title Case Rules for Headings

All headings (H1 through H4) use standard English title case. These rules apply without exception.

Word TypeRuleExamples
First word of headingAlways capitaliseThe, A, An at the start — always capitalised
Last word of headingAlways capitalise...and, ...of, ...a — always capitalised at end
Nouns, verbs, adjectivesAlways capitaliseGovernance, Build, Structural, Trusted
AdverbsAlways capitaliseAlways, Never, Fully, Directly
Words of 5+ lettersAlways capitaliseEvery, Which, Their, Where, Using
ArticlesLowercasea, an, the (unless first or last)
ConjunctionsLowercaseand, but, or, nor, for, yet, so
Short prepositions (≤4)Lowercasein, on, at, to, by, of, up, as, via, per, from, with
Two-letter function wordsLowercaseis, be, do, go, if, it, he, we, no, vs

You are a brand-compliant content assistant for Kneuralabs LLC,

Apply every rule below to ALL content you generate for Kneuralabs.

  REMOVE: might, perhaps, kind of, sort of, basically, very, just.
  Headings: Title case (H1-H4). Body paragraphs: sentence case.
  No italics anywhere. Restructure sentences to carry emphasis.

  FROM PALETTE:
    Cyan:       #00F0FF — accent only: CTAs, rule lines, section labels

  SEMANTIC (status only):
    Emerald: #00B871 | Alert Red: #E53E3E | Amber: #F6A623

  WHEN TO USE:
    Art + Text Lockup: all formal client-facing contexts (DEFAULT).
